Introducción a la Expo :

Expo es un marco y una plataforma para aplicaciones universales de React. Es un conjunto de herramientas y servicios creados alrededor de React Native y plataformas nativas que lo ayudan a desarrollar, construir, implementar e iterar rápidamente en iOS, Android y aplicaciones web desde la misma base de código JavaScript / TypeScript.

Preguntas comunes :

menu de preguntas

¿Expo es similar a React para el desarrollo web?

Expo y React Native son similares a React. Tendrá que aprender un nuevo conjunto de componentes (en View lugar de div, por ejemplo) y escribir aplicaciones móviles es muy diferente de los sitios web; usted piensa más en términos de pantallas y navegadores diferentes en lugar de páginas web separadas, pero se transfiere mucho más de su conocimiento que si estuviera escribiendo una aplicación tradicional de Android o iOS.

¿Cuánto cuesta la Expo?

Las herramientas de la exposición son de código abierto y de uso gratuito. Puede ver la fuente de Expo SDK y el cliente en https://github.com/expo/expo

¿Cómo Gana dinero si Expo es gratuita?

El hecho de que pueda ejecutar todo lo que proporciona Expo en su propio hardware no significa que configurar y administrar todos los servicios que necesita sea lo más adecuado para su equipo. Si eso le suena a usted, puede ver lo que ofrecemos en https://expo.io/developer-services

¿Cómo agrego código nativo personalizado a mi proyecto de Expo?

Los proyectos de Managed Expo no admiten código nativo personalizado, incluidas las bibliotecas de terceros que requieren componentes nativos personalizados. En un proyecto administrado, solo escribe JavaScript. En nuestro SDK , le ofrecemos un gran conjunto de módulos nativos de alta calidad comúnmente deseados. Recomendamos hacer tanto en JavaScript como sea posible, ya que se puede implementar inmediatamente para todos sus usuarios y funcionar en ambas plataformas, y siempre continuará beneficiándose de las actualizaciones de Expo SDK. Sin embargo, si necesita algo personalizado que no es posible con los módulos nativos proporcionados en el SDK, como procesamiento de video sobre la marcha o control de bajo nivel sobre la radio Bluetooth para realizar una actualización de firmware y otras funciones solicitadas aquí , puede ejecutar expo ejecty tener control total sobre los proyectos nativos subyacentes.

¿Cómo hago para que mi proyecto existente de React Native se ejecute con Expo?

En este momento, la forma más fácil de hacer esto es usar expo init(con Expo CLI) para crear un nuevo proyecto y luego copiar todo el código fuente JavaScript de su proyecto existente, y luego yarn addlas dependencias de la biblioteca que tiene. Si tiene dependencias de módulos nativos similares a lo que se expone a través del Expo SDK, este proceso no debería tomar más de unos minutos (sin incluir el npm installtiempo). No dude en hacernos preguntas si tiene algún problema. Si está utilizando bibliotecas nativas que no son compatibles con Expo, tendrá que reescribir algunas partes de su aplicación para usar API similares que son parte de Expo, o es posible que no pueda obtener todas las partes de su aplicación para trabaja. ¡Sin embargo, hay muchas cosas!

¿Qué versión de Android e iOS son compatibles con las aplicaciones de Expo?

Expo es compatible con Android 5+ e iOS 11+.

Presentando JSX:

Considera la declaración de esta variable:

					
						const element = 
Hello, world!
;

Esta curiosa sintaxis de etiquetas no es ni un string ni HTML.

Se llama JSX, y es una extensión de la sintaxis de JavaScript. Recomendamos usarlo con React para describir cómo debería ser la interfaz de usuario. JSX puede recordarte a un lenguaje de plantillas, pero viene con todo el poder de JavaScript.:

React acepta el hecho de que la lógica de renderizado está intrínsecamente unida a la lógica de la interfaz de usuario: cómo se manejan los eventos, cómo cambia el estado con el tiempo y cómo se preparan los datos para su visualización. En lugar de separar artificialmente tecnologías poniendo el maquetado y la lógica en archivos separados, React separa intereses con unidades ligeramente acopladas llamadas “componentes” que contienen ambas. Volveremos a los componentes en otra sección, pero si aún no te sientes cómodo maquetando en JS, esta charla podría convencerte de lo contrario.:

Insertando expresiones en JSX:

					
						const name = 'Josh Perez';
						const element = 
Hello, {name}
;

						ReactDOM.render(
  						element,

Puedes poner cualquier expresión de JavaScript dentro de llaves en JSX. Por ejemplo, 2 + 2, user.firstName, o formatName(user) son todas expresiones válidas de Javascript.

En el ejemplo a continuación, insertamos el resultado de llamar la función de JavaScript, formatName(user), dentro de un elemento.

Declarativo

React te ayuda a crear interfaces de usuario interactivas de forma sencilla. Diseña vistas simples para cada estado en tu aplicación, y React se encargará de actualizar y renderizar de manera eficiente los componentes correctos cuando los datos cambien.

Basado en componentes

Crea componentes encapsulados que manejen su propio estado, y conviértelos en interfaces de usuario complejas.
